# Manipuri Stemmer for Meitei Mayek Script 
 
A grapheme-aware, rule-based stemmer for Manipuri (Meiteilon) that operates on Unicode grapheme clusters. 
 
## Features 
- Longest-match prefix removal 
- Iterative suffix stripping 
- Grapheme-aware processing for lonsum characters 
- Curated inventory: 6 prefixes and 156 suffixes 
 
## Evaluation Results 
- Accuracy: 92.82%% 
- Precision: 92.82%% 
- Recall: 92.82%% 
- F-measure: 92.82%%
